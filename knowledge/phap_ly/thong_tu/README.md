# Thông tư — chưa có nội dung

## OPEN QUESTION
Chưa xác nhận có Thông tư hướng dẫn nào (Bộ Xây dựng, Bộ Tài chính...) cần đưa vào Knowledge Graph cho bản demo hay không — xem `docs/07_KNOWLEDGE_GRAPH.md`.

## Quy ước khi có nội dung
Mỗi Thông tư tạo 1 file riêng trong thư mục này, đặt tên `thong_tu_<so_hieu>.md`, theo cùng cấu trúc với các file trong `../nghi_dinh/` (Metadata → Nội dung sửa đổi chính → Quan hệ trong Knowledge Graph → OPEN QUESTION nếu có).

Không tạo file Thông tư giả định trước khi có xác nhận cụ thể.
